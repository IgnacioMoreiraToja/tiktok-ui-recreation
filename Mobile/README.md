# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

3. Make sure you've created a .env file where you can export your device's URL. It has to be named EXPO_PUBLIC_API_URL. If you plan to use the app on an external device, it has to refer to your machine's IP address (you can see this in your WIFI Settings or in the terminal when you run npx expo start). In api.js file you have to replace the value of urlApi with EXPO_PUBLIC_API_URL 

4. Note: Sometimes we have to segment the data that the API provides to us, to get a better performance in the APP.

5. Enjoy :D.
