# TikTok UI Recreation

A user interface recreation inspired by the TikTok video feed.

This project was built as part of a university UI course to explore modern mobile interface patterns.

## Technologies

- React
- React Native
- CSS

## Features

- Vertical video feed layout
- Mobile-first interface design
- UI component structure using React

## Purpose

This project focuses on interface design and layout recreation, not on replicating TikTok functionality.
